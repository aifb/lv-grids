# lv-grids
Test cases of LV distribution grids


## Technical Information

* IEEE common data format (see http://www2.ee.washington.edu/research/pstca/formats/cdf.txt)


## Overview

* <i>lv-grid-1.txt</i>: sub-urban LV distribution grid
